# 📝 Flask To-Do App

تطبيق مهام بسيط تم تطويره باستخدام:
- *Flask* (باك إند)
- *SQLite* (قاعدة بيانات)
- *HTML / CSS / JavaScript* (فرونت إند)

## ⚙️ المميزات
- إضافة مهمة جديدة  
- حذف مهمة  
- تغيير حالة المهمة (مكتملة / غير مكتملة)  
- تحديث فوري بدون إعادة تحميل الصفحة  
- تصميم بسيط وأنيق

## 🚀 طريقة التشغيل
```bash
git clone https://github.com/marwahamdy/todo-app-flask.git
cd todo-app-flask
pip install -r requirements.txt
flask run/
